# Deep-Dive-C

This is my personal C programming practice repository.  
It contains only the code examples and exercises I solve while learning C.

## Topics

- 01-Basics → Variables, data types, operators
- 02-Control-Statements → if, else, switch
- 03-Loops → for, while, do-while, break/continue, goto
- 04-Functions → Functions and recursion
- 05-Arrays → 1D and 2D arrays
- 06-Pointers → pointers, pointer arithmetic, void, NULL, dangling, wild
- 07-Strings → string handling, string.h functions
- 08-Structures-Unions → struct, typedef, union
- 09-Dynamic-Memory → malloc, calloc, realloc, free
- 10-File-Handling → file reading/writing exercises
- 11-Preprocessor → #define, #include, predefined macros
- 12-Advanced-Concepts → function pointers, callbacks, memory leaks

## Notes

- Each file should be named clearly, e.g., `for_loop_sum.c`  
- Add comments inside code for clarity  
- Push code regularly after practicing
